# Test Task

React + Vite + TypeScript проект с компонентной версткой таблицы пользователей.

## Технологии

- React 19
- TypeScript
- Vite
- SCSS
- React Hooks

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Предпросмотр

```bash
npm run preview
```

## Структура проекта

```
src/
├── assets/          # Иконки и изображения
├── components/      # React компоненты
├── data/           # Моковые данные
├── hooks/          # Кастомные хуки
├── styles/         # Глобальные стили и переменные
└── types/          # TypeScript типы
```
